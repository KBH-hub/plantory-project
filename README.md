# 🌱 프로젝트명
식물 나눔 및 관리 웹 리팩토링 프로젝트

# 📌 서비스 소개 
**식물 지역 나눔 커뮤니티 및 식물 관리 프로그램**
- 식물을 주제로 한 커뮤니티 - 지역 무료 식물 나눔 커뮤니티, 질문 커뮤니티 
- 식물 사전 - 실내 식물 사전, 건조 식물 사전/ 나의 식물 - 식물 캘린더, 내 식물 관리

# 🛠 기술 스택
| 구분 | 기술 스택 |
|---|---|
| **OS** | Ubuntu (AWS EC2) |
| **WAS** | Apache Tomcat 10 |
| **Front** | JavaScript, Axios |
| **Back** | Java, Spring Boot, MyBatis, Maven, JUnit 5, Thymeleaf, Spring Security |
| **Database** | MariaDB, Google Cloud Service |
| **형상관리** | Git, GitHub |
| **Tooling** | IntelliJ, Postman, ERD Cloud, Figma, draw.io, Jira, Confluence |
